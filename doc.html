Game.js

A simple JavaScript game engine.

Core Principles

  • Work with Vanilla JavaScript without needing a webserver server.
  • Keep it simple.

Features

  • Sprites
  • Backgrounds
  • Sounds
  • Objects
  • Rooms
  • Collision (AABB)
  • Tile based collision

Object Referrence

Game

A singleton object which controls the game.

Methods

Game.start(options)

Starts the game.
